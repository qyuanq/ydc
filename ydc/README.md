# About

>此项目是vue+node（koa）+mysql构建的自媒体发布平台

# 技术栈
>vue2 + fetch + vue-router + webpack + ES6/7 + echart + vue-quill-editor + koa + mysql

# 项目运行
  git clone https://github.com/qyuanq/ydc.git
  
## 前端启动
  cd ydc/ydc
  
  npm install
  
  npm run start
  
  访问：http://localhost:8080
  
## 服务端启动
  cd ydc/server
  
  npm install
  
  node server
